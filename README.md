# template react

## 简介

react 项目模版。附带完整工程化方案。

## 运行

```bash
# 环境准备
npm i -g pnpm

# 安装依赖
pnpm i

# 启动
pnpm dev
```

## 方案选择

- 包管理器
  - [pnpm](https://github.com/pnpm/pnpm)
- 编译
  - [vite](https://github.com/vitejs/vite)
- 工程化
  - [prettier](https://github.com/prettier/prettier)
